How to Use Chrome DevTools Device Toolbar Open DevTools Right-click anywhere on the page and select Inspect, or press Ctrl+Shift+I (Windows/Linux) or Cmd+Option+I (Mac). Toggle Device Toolbar Click the Toggle Device Toolbar icon (ðŸ“±) in the top-left corner of DevTools, or press Ctrl+Shift+M / Cmd+Shift+M. Choose a Device Use the dropdown at the top to select a device like iPhone 12, Pixel 5, iPad, etc. You can also add custom devices by clicking Edit in the dropdown. Rotate the Screen Click the rotate icon (ðŸ”„) to switch between portrait and landscape modes. Set Zoom and Resolution Adjust the zoom level and resolution to test responsiveness. You can also simulate different DPR (device pixel ratio) settings. Test Touch Events The toolbar simulates touch events, so you can test gestures like swipe and pinch. Network Throttling Use the Network tab to simulate slower connections (e.g., 3G) to test loading performance. Capture Screenshots Right-click on the page and choose Capture screenshot or use the command menu (Cmd+Shift+P) and search for screenshot options.
